## 📊 DATABASE SCHEMA

### 6 Core Tables
1. **food_categories** - Menu categories
2. **food_items** - Food items with options
3. **restaurants** - Restaurant info
4. **orders** - User orders
5. **order_items** - Items in orders
6. **user_profiles** - User information

### Relationships
- Food items → Categories (many-to-one)
- Food items → Restaurants (many-to-one)
- Orders → Users (many-to-one)
- Order items → Orders (many-to-one)
- Order items → Food items (many-to-one)
